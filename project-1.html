Aplicación de Gestión de Personas


La aplicación de Gestión de Personas es una solución desarrollada con Angular en el frontend y Spring Boot en el backend, utilizando PostgreSQL como base de datos. Esta aplicación permite a los usuarios agregar, editar y eliminar registros de personas, proporcionando una interfaz amigable y fácil de usar para la gestión de datos

Detalles

Frontend: Angular 17
Implmentación de servicios para realizar peticiones HTTP al backend
Estilos para una interfaz mas amigable
Validacion de formulario para el envio completo de datos
backend: SpringBoot 3.3.2
Arquitectura RESTful para manejar las operaciones CRUD
Implmentación de Controladores, Servicios y Repositorio
Base de Datos: PostgreSQL
Diseño de tablas para almacenar la información de las personas
Configuración de la conexión a la base de datos en Spring Boot.
